About the project

RemedySense is a college project that uses deep learning for skin issue detection and a rule-based + collaborative filtering recommendation engine to suggest natural remedies. Tech stack: Python, TensorFlow/PyTorch, Flask, HTML/CSS/JS.

How it works

  1. Upload a skin image or enter details (age, skin type, issue).
  2. The model classifies the problem (acne, dryness, pigmentation, etc.).
  3. The recommender matches remedies from a curated knowledge base and personalizes suggestions.
