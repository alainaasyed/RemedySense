The Prediction Engine: Decision Tree

How our model translates visual data into therapeutic needs for personalized recommendations.

Why a Decision Tree?

We chose the Decision Tree Model for its **interpretability** and its ability to handle complex, non-linear relationships found in image features like color, texture, and variance. This allows for clear, justifiable predictions.

Performance & Features

  • Accuracy: 92% (for property prediction)
  • Key Features: Color Variance, Texture Roughness, Area Size (primary inputs from image analysis)
